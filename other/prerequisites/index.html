Orbit

Prerequisites

Required Software

The minimum requirements for Orbit are:

  • Java 8 SE SDK or above.
  • Maven 3.1 or above.

Recommended Software

  • Git is required if you want to get and build the Orbit code manually.
  • IntelliJ IDEA is our preferred IDE.
